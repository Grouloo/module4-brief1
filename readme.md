# Module 4 Brief 1

## Structure du dépôt

- **notebook.ipynb :** Notebook de nettoyage des donées et évaluation des modèles
- **/data :** Contient les datasets bruts, nettoyés, et conformes au format CSV

## Procédure

Le notebook contient la procédure suivante :

1. Étude du jeu de données
2. Nettoyage du jeu de données
3. Mise en conformité
4. Standardisation
5. Train-test split
6. Entraînement de 3 modèles
7. Cross-validation des 3 modèles
